The bulk of this repository consists of programming languages, and they are

  • Python, a dynamically typed, garbage-collected, high-level general purpose language

    which emphasizes code readibility, as outlined in the Zen of Python;

  • C and C++, high-level languages designed for low-level access to memory. C is imperative and procedural,

    while C++, a superset of C, is object-oriented;

  • Java, an object-oriented language, primarily defined by (its verbosity, and) the motto Write Once, Run Anywhere,

    as it was designed to run on any device with a Java Runtime Environment, which most devices nowadays do;
